# WealthTech Job Search

AI-powered wealthtech GTM & commercial leadership job search agent built with Next.js and the Anthropic API.

## Setup

1. Install dependencies: `npm install`
2. Create `.env` from `.env.example` and add your Anthropic API key
3. Run locally: `npm run dev`

## Vercel Deployment
